## 📋 拷贝即用部署步骤

当前可用的部署流程:

### 1. 拷贝项目文件
```
复制整个 deepcad 文件夹到目标计算机
```

### 2. 环境检查
```batch
双击运行: check_env.bat
```

### 3. 一键安装
```batch
双击运行: setup.bat
```

### 4. 启动服务 (后端)
```batch
python -m uvicorn gateway.main:app --host 0.0.0.0 --port 8000 --reload
```

### 5. 前端开发模式 (临时方案)
```batch
cd frontend
npm run dev
```
