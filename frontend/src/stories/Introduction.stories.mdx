# DeepCAD 组件库

欢迎使用 DeepCAD 深基坑CAE分析平台组件库！这是一个专业的工程分析软件组件集合，
专为现代Web应用程序设计，提供了完整的CAE工作流程支持。

## 🌟 平台特色

### 🎯 专业级CAE分析
- 深基坑工程专业分析
- 多物理场耦合计算
- 高精度数值模拟
- 实时结果可视化

### 🎨 现代化设计系统
- 毛玻璃视觉效果
- 响应式布局设计
- 无障碍支持
- 暗色模式适配

### 🔧 完整工作流程
- DXF文件导入
- 几何建模工具
- 网格生成系统
- 材料属性管理
- 边界条件设置
- 求解器配置
- 后处理分析

## 📚 组件分类

### UI 基础组件
- **Glass Components** - 现代毛玻璃效果的基础UI组件
- **Form Controls** - 基于react-hook-form的表单控件
- **Layout Components** - 布局和导航组件

### 业务组件
- **DXF Import** - DXF文件导入和处理
- **3D Viewport** - 三维可视化视口
- **Meshing Tools** - 网格生成和管理
- **Material Library** - 材料库管理
- **Analysis Tools** - 分析工具集

### 专业模块
- **Geology Module** - 地质建模模块
- **Excavation Module** - 基坑开挖模块
- **Support Module** - 支护结构模块

## 🚀 快速开始

### 安装依赖

```bash

npm install

# 使用pnpm
pnpm install
```

### 启动开发服务器

```bash
# 启动前端开发服务器

# 启动Storybook

```

```bash

cd gateway

python start_backend.py
```

## 📖 使用指南

### 1. 组件导入

```tsx
import { GlassCard, GlassButton } from '@/components/ui/GlassComponents';
import { FormInput, FormSelect } from '@/components/forms/FormControls';
```

### 2. 基础使用

```tsx
function MyComponent() {
  return (
    <GlassCard variant="elevated" size="md">
      <h2>CAE分析结果</h2>
      <FormInput 
        name="projectName"
        label="项目名称"
        placeholder="请输入项目名称"
      />
      <GlassButton variant="primary">
        开始分析
      </GlassButton>
    </GlassCard>
  );
}
```

### 3. 表单集成

```tsx
import { useForm } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';
import { z } from 'zod';

const schema = z.object({
  projectName: z.string().min(1, '项目名称不能为空'),
  soilType: z.string().min(1, '请选择土层类型'),
});

function AnalysisForm() {
  const form = useForm({
    resolver: zodResolver(schema),
  });

  return (
    <Form form={form} onSubmit={console.log}>
      {/* 表单字段 */}
    </Form>
  );
}
```
