# 设计系统

DeepCAD设计系统是一套完整的视觉语言和组件规范，旨在为专业CAE分析平台
提供统一、现代、高效的用户体验。

## 🎨 设计理念

### 专业性 (Professional)
- 面向工程师和专业用户
- 清晰的信息层次结构
- 精确的数据展示
- 专业的工具集成

### 现代性 (Modern)
- 毛玻璃视觉效果
- 流畅的动画过渡
- 响应式设计
- 暗色模式支持

### 高效性 (Efficient)
- 快速的操作流程
- 直观的用户界面
- 智能的默认设置
- 强大的快捷键支持

## 🔘 组件规范

### 按钮系统

按钮是用户交互的主要元素，遵循以下层次结构：

1. **Primary Button** - 主要操作按钮
2. **Secondary Button** - 次要操作按钮
3. **Ghost Button** - 辅助操作按钮
4. **Outline Button** - 边框按钮

### 表单控件

表单控件保持一致的视觉样式和交互行为：

- 统一的焦点状态
- 一致的错误提示样式
- 清晰的标签和说明文本
- 合适的触摸目标大小

### 卡片组件

卡片是内容组织的基本单元：

- **Default** - 标准卡片样式
- **Elevated** - 带阴影的悬浮效果
- **Outline** - 带边框的轮廓样式

## 🌊 动画系统

### 过渡时间

```css
/* 快速过渡 - 用于简单状态变化 */
transition-duration: 150ms;

/* 标准过渡 - 用于大多数交互 */
transition-duration: 200ms;

/* 慢速过渡 - 用于复杂动画 */
transition-duration: 300ms;
```

### 缓动函数

```css
/* 标准缓动 */
transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);

/* 进入缓动 */
transition-timing-function: cubic-bezier(0, 0, 0.2, 1);

/* 退出缓动 */
transition-timing-function: cubic-bezier(0.4, 0, 1, 1);
```

## 📱 响应式设计

### 断点系统

```css
/* 移动端 */
@media (max-width: 640px) { /* sm */ }

/* 平板端 */
@media (min-width: 641px) and (max-width: 1024px) { /* md-lg */ }

/* 桌面端 */
@media (min-width: 1025px) { /* xl+ */ }
```

### 布局原则

- **移动优先** - 从小屏幕开始设计
- **内容适配** - 内容驱动布局变化
- **性能考虑** - 减少不必要的布局重排

## ♿ 无障碍设计

### 色彩对比度

所有文本和背景色组合都满足WCAG 2.1 AA级标准：

- 正常文本：对比度 ≥ 4.5:1
- 大文本：对比度 ≥ 3:1

### 键盘导航

- 所有交互元素都支持键盘访问
- 清晰的焦点指示器
- 合理的Tab键顺序

### 屏幕阅读器

- 语义化的HTML结构
- 适当的ARIA标签
- 有意义的替代文本

## 🎯 最佳实践

### 组件设计

1. **单一职责** - 每个组件只负责一个功能
2. **可复用性** - 组件应该易于在不同场景下使用
3. **可访问性** - 遵循无障碍设计原则
4. **性能优化** - 避免不必要的重渲染
