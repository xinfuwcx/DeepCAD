## 后续开发建议

### 桩锚体系参数设计
建议为桩锚体系添加以下参数：
- 桩基参数：桩径、桩长、桩距、材料等级
- 锚索参数：锚索长度、预应力、倾角、间距
- 冠梁参数：尺寸、配筋、连接方式
- 施工参数：施工顺序、质量要求等

### Schema 扩展
可以在 `schemas/index.ts` 中添加新的 `PileAnchorSchema` 来定义桩锚体系的参数验证规则。
